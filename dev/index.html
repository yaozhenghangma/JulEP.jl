MatterEnv

MatterEnv is an atomic environment package embedded in Julia. It defines basic type to describe atomic environment and provides postprocessing tools for DFT calculation.

Features

  • Plot band structure and projected band structure
  • Generate and plot electronic density of states (DOS) and projected DOS
  • VASP file I/O
